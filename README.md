## 🔊 프로젝트 소개
COVID-19 이후 디지털 워크 성장에 발맞춰 온라인을 통해서 효율적인 업무 수행과 팀 간 원활한 소통을 지원하기 위한 종합적인 협업툴 서비스입니다.
해당 서비스는 분산된 팀과 원격 근무 환경에서도 seamless한 협업을 가능하게 하여 조직의 생산성을 증진시킵니다.

## 🕒 프로젝트 기간
2024.05.03 ~ 2024.06.10까지 진행 후 리팩토링하는 작업을 거쳤습니다.
